Format CSV

Please use the following column order in your CSV file.

  1. Name
  2. Address
  3. City
  4. Zipcode
  5. State
  6. Country code (ISO standard)
  7. Main phone number
  8. Description (HTML allowed)

Note: First line of CSV file must not contain location details since it's skipped by this tool. Use it for column titles.

Fill in your location details


For the description you can use variables mentioned below click to insert). The variables will be replaced by the corresponding values of this form.
{name}, {address}, {city}, {zipcode}, {state}, {country}, {phone}
